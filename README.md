# 🚫 No YouTube Shorts

A Chrome extension that completely removes YouTube Shorts with a simple toggle to enable or disable whenever you want.

---

## 📌 About the Project

YouTube has been increasingly pushing short-form content (Shorts), which can be highly distracting and harmful to focus.

This project solves that problem in a simple way:

- ❌ Removes Shorts from the interface  
- 🎯 Keeps only long-form videos  
- 🔘 Includes an ON/OFF toggle via popup  

The goal is to **reduce distractions and improve the YouTube experience**.

---

## ⚙️ Features

- Removes:
  - Shorts from home feed  
  - Shorts from recommendations  
- Lightweight and fast (runs locally, no backend)
- Toggle button to enable/disable anytime

---

## 🧠 How It Works

The extension manipulates the YouTube DOM, removing elements related to Shorts in real time.

Since YouTube does not provide a native way to disable Shorts, this approach ensures full control over what is displayed.

---

## 🛠️ Tech Stack

- JavaScript (Vanilla)
- HTML
- CSS
- Chrome Extensions (Manifest V3)
